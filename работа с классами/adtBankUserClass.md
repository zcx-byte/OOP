# ADT BankUserClass

• **Данные**
  * name — имя пользователя (строка)
  * bank_ID — идентификатор банка, к которому привязан пользователь (строка)
  * balance — текущий баланс пользователя в банке (вещественное число)

• **Операции**
  
  – **Конструктор**
    Вход: name (строка), bank_ID (строка), initial_balance (вещественное число)
    Предусловия: initial_balance >= 0
    Процесс: инициализация полей объекта переданными значениями
    Выход: -
    Постусловия: создан объект с заданным именем, идентификатором банка и начальным балансом
    Исключения: std::invalid_argument если initial_balance < 0
  
  – **Операция «Получить баланс»**
    Вход: bank_ID (строка)
    Предусловия: отсутствуют (параметр bank_ID не проверяется в текущей реализации)
    Процесс: возврат текущего баланса
    Выход: значение баланса (вещественное число)
    Постусловия: состояние объекта не изменяется
  
  – **Операция «Получить имя пользователя»**
    Вход: bank_ID (строка)
    Предусловия: отсутствуют (параметр bank_ID не проверяется в текущей реализации)
    Процесс: возврат имени пользователя
    Выход: имя пользователя (строка)
    Постусловия: состояние объекта не изменяется
  
  – **Операция «Пополнить счет»**
    Вход: amount (вещественное число), bank_ID (строка)
    Предусловия: amount > 0
    Процесс: увеличение баланса на amount
    Выход: -
    Постусловия: баланс увеличен на amount
    Исключения: std::invalid_argument если amount <= 0
    Примечание: Параметр bank_ID не проверяется в текущей реализации
  
  – **Операция «Снять средства»**
    Вход: amount (вещественное число), bank_ID (строка)
    Предусловия: 
    * amount > 0
    * balance >= amount
    Процесс: уменьшение баланса на amount
    Выход: -
    Постусловия: баланс уменьшен на amount
    Исключения: 
    * std::invalid_argument если amount <= 0
    * std::runtime_error если средств на счете недостаточно
    Примечание: Параметр bank_ID не проверяется в текущей реализации
  
  – **Операция «Изменить имя пользователя»**
    Вход: new_name (строка), bank_ID (строка)
    Предусловия: отсутствуют
    Процесс: изменение имени пользователя на new_name
    Выход: -
    Постусловия: имя пользователя изменено
    Примечание: Параметр bank_ID не проверяется в текущей реализации
  
  – **Конструктор копирования**
    Вход: other (существующий объект BankUserClass)
    Предусловия: -
    Процесс: создание нового объекта путем копирования значений полей из other
    Выход: -
    Постусловия: создан новый объект с такими же значениями полей, как у other
  
  – **Операция «Сохранить в файл (режим дозаписи)»**
    Вход: filename (строка) - имя файла для сохранения
    Предусловия: файл должен быть доступен для записи
    Процесс: открытие файла в режиме дозаписи и запись данных объекта (name, bank_ID, balance) в одну строку
    Выход: -
    Постусловия: данные объекта сохранены в файл
    Исключения: std::ios_base::failure если файл не удалось открыть
  
  – **Операция «Загрузить из файла»**
    Вход: file (поток ifstream) - открытый файловый поток для чтения
    Предусловия: поток должен быть открыт и готов к чтению
    Процесс: чтение данных из потока в порядке: name, bank_ID, balance
    Выход: -
    Постусловия: поля объекта перезаписаны значениями из файла
    Исключения: std::ios_base::failure если чтение из потока невозможно
  
  – **Операция «Получить строковое представление»**
    Вход: -
    Предусловия: -
    Процесс: формирование строки с информацией об объекте
    Выход: строка вида "Name: ..., Bank ID: ..., Balance: ..."
    Постусловия: состояние объекта не изменяется
